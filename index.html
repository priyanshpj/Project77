Project on console

Visual studio is a coding platform which is used to develop computer programs, websites and apps.

Console screen accepts input parameters, calls the required service, runs buisness logic and sends output to the console.To open the console screen, press Ctrl + Shift + i and go to the Console tab.To print anything on the console screen, type console.log() and enter a variable name in the brackets or open a double-quote and write a string in it.
